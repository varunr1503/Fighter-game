## Gameplay

- **2 Players** on the same keyboard
- Retro-style combat with pixel graphics and special attacks

### Controls

#### Player 1:
- Movement: `W`, `A`, `S`, `D`
- Attacks: `E` (Attack 1), `R` (Attack 2)

#### Player 2:
- Movement: Arrow Keys (`‚Üë ‚Üì ‚Üê ‚Üí`)
- Attacks: `N` (Attack 1), `M` (Attack 2)
